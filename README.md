# Doge's Keyboard Collection

A new game website for people who want a chill and relaxing time while at work, studying or just in class. This is the remaster of the original [google site](https://sites.google.com/view/dogeskeyboardcollection). The original was just made for fun but this has a remastered UI and would probably be updated more regularly than the other one. Free for anyone to use at all.
